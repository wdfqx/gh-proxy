GitHub 文件加速

GitHub文件链接带不带协议头都可以，支持release、archive以及文件，右键复制出来的链接都是符合标准的。
文件使用cf加速。注意：不支持项目目录

本项目支持clone公开和私有仓库，用法示例：
对于公开仓库的用法： git clone https://ghproxy.com/https://github.com/username/project.git
对于私有仓库的用法： git clone user:token@ghproxy.com/https://github.com/username/project.git

分支源码：https://github.com/username/project/archive/master.zip
release源码：https://github.com/username/project/archive/v0.1.0.tar.gz
release文件：https://github.com/username/project/releases/download/v0.1.0/example.zip
分支文件：https://github.com/username/project/blob/master/filename
